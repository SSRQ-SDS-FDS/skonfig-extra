cdist-type__consul_reload(7)
============================

NAME
----
cdist-type__consul_reload - Reload consul


DESCRIPTION
-----------
Reload consul after configuration changes.


REQUIRED PARAMETERS
-------------------
None.


OPTIONAL PARAMETERS
-------------------
None.


EXAMPLES
--------

.. code-block:: sh

    __consul_reload
