cdist-type__docker_compose(7)
=============================

NAME
----
cdist-type__docker_compose - install docker-compose


DESCRIPTION
-----------
Installs latest docker-compose package from dockerproject.org.


REQUIRED PARAMETERS
-------------------
None.


OPTIONAL PARAMETERS
-------------------
None.


BOOLEAN PARAMETERS
------------------
None.


EXAMPLES
--------

.. code-block:: sh

    __docker_compose
